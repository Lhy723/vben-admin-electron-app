## 项目定位

本项目是基于 Vue Vben Admin 的 Electron 桌面应用定制版本，深度融合 Electron 与 Web 技术，提供完整的跨平台桌面应用解决方案。

## 核心优势

- **Electron 深度集成**：支持文件系统操作、原生窗口控制、系统通知等桌面应用特性
- **全平台构建**：支持 Windows/macOS/Linux 三大平台安装包构建
- **现代技术栈**：基于 Vue3 + Vite + TypeScript + TailwindCSS 开发框架
- **完整功能包**：包含多主题、国际化、动态权限系统等核心功能
